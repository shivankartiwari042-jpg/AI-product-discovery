# AI-Powered Product Discovery Platform

## Project Description
An AI shopping assistant that aggregates product data from multiple websites and helps users discover, compare, and choose products through natural language conversations.

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: FastAPI
- Database: PostgreSQL
- Web Scraping: BeautifulSoup + Selenium
- AI: spaCy + Sentence Transformers
- Semantic Search: FAISS/ChromaDB
- Deployment: Docker
